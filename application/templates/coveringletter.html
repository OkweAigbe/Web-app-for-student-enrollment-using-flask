This project is for student enrollment into courses listed by HTA. uses html, css, boostraps and python flask framework and MongoDB database engine. Please see requirements.txt file for other libraries that will be needed. An Object oriented approach was used and few api codes were written for testing purpose. Please see routes.py files.
